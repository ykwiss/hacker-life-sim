# 黑客人生模拟器 · 自由沙盒版 v2.0

该原型使用 Python + Tkinter 构建，围绕“完全自由、系统驱动、动态世界”的理念实现了一个黑客人生沙盒的可视化入口。界面强调**阶段式解锁**：玩家不会一开始就看到全部功能，而是随着行动逐渐获取训练、市场、任务、补给等模块，配合常驻终端与赛博风格营造真实的“操控台”感。

## 功能亮点

- **模块化内容**：背景、训练课程、任务契约、装备清单都存放在 `hacker_sim/data.py` 中，以易于扩展的结构组织，后续添加新装备、新任务只需追加数据即可。
- **沙盒循环**：流程从“接入→训练→任务市场→自由沙盒”，每个阶段都会更新指令与可用操作，避免信息过载。
- **风险与危机系统**：执法关注/市场波动会触发危机卡，玩家需要在弹窗中选择方案，通过技能或装备获得额外成功率加成。
- **动态市场**：一键刷新全球态势，合法/地下市场倍率与执法压力实时变化，直接影响报酬与风险计算。
- **常驻终端**：所有事件都会流入终端日志，玩家像操作真实黑客面板一样关注状态、指令与危机提示。
- **弹出式模块**：训练、任务、补给均在独立面板中完成，便于后续接入更复杂的界面、小游戏、剧情或实时倒计时。
- **成长感**：角色从 10 岁起步，系统根据行动事件自动成长，终端会记录每一次年龄突破。

## 快速体验

```bash
cd hacker_sandbox
python3 src/main.py
```

1. 左侧“接入配置”输入代号并选择背景，点击“接入节点”。
2. 首先进入“训练实验室”并完成一门课程，界面会自动提示刷新市场。
3. 刷新后可访问“任务集市”接取合法/地下契约；完成首个任务后即进入完全自由阶段，可继续训练、刷市场、购买装备。
4. 当执法关注或市场波动达到阈值时，会自动弹出“危机响应”面板，依据技能/人脉选择方案。
5. 常驻终端输出每一次训练、任务、危机、购买与市场事件，帮助沉浸式记录你的黑客人生。

## 架构概览

```
hacker_sim/
├── data.py        # 背景、训练、任务、装备、市场模板
├── engine.py      # 状态管理、概率计算、市场与任务逻辑
├── models.py      # 数据模型（Player、TrainingModule 等）
└── ui.py          # 阶段式 Tkinter UI + 终端/弹出面板
```
